Overview
--------
This repository contains a simple static portfolio site showcasing projects, maps, code, and data related to geospatial analysis, GeoAI, and climate-focused work.

Local preview
-------------
Open `index.html` in your browser to preview locally (no build step required):

1. In File Explorer, double-click `index.html`, or
2. From a simple HTTP server (recommended for some browser features):

```powershell
# Windows (PowerShell)
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Push to GitHub
--------------
1. Create a new repository on GitHub (e.g., `portfolio`).
2. Then run these commands locally (replace URL with your repo URL):

```powershell
git remote add origin https://github.com/<your-username>/<repo>.git
git branch -M main
git push -u origin main
```
